# Multiplication-Table-Generator.py

Master Loops by Creating a Multiplication Table Generator (Basic Level)

To evaluate your understanding of for loops, while loops, and iteration logic in Python by building a program that generates multiplication tables dynamically.

Simple README: How it Works 

•  while makes the program stay open. 

•  for does the math over and over again. 

•  try keeps the program from breaking if someone makes a typing mistake.
